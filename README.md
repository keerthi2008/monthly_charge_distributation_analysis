# monthly_charge_distributation_analysis

## 📌 Project Overview

This project focuses on analyzing customer churn data to understand why customers leave a telecom service. Using Excel, we explore patterns, trends, and key factors that influence churn, and provide actionable insights to improve customer retention.

---

## 🎯 Objectives

* Analyze churned vs non-churned customers
  
* Identify key factors affecting churn
  
* Study Monthly Charges distribution
  
* Evaluate impact of services on churn
  
* Build an interactive dashboard

---

## 📂 Dataset

* **Name:** Telco Customer Churn Dataset
  
* **Format:** CSV
  
* **Rows:** ~7000 customers
  
* **Columns:** Customer demographics, services, billing, churn status

---

## 🛠 Tools Used

* Microsoft Excel
  
* Pivot Tables
  
* Charts (Column, Pie, Histogram, Scatter Plot)
  
* Data Cleaning Techniques

---

## 🔄 Data Cleaning Steps

* Removed blank/null values
  
* Converted text to numerical format (Yes = 1, No = 0)
  
* Checked for duplicates
  
* Ensured correct data types
  

---

## 📊 Analysis Performed

### 1️⃣ Churn Count Visualization

* Created column chart to compare churn vs non-churn
  
* Added data labels and formatted chart

---

### 2️⃣ Monthly Charges Distribution

* Created histogram for MonthlyCharges
  
* Used bin size = 10
  
* Analyzed distribution of customers across price ranges

---

### 3️⃣ Services Impact on Churn

* Used pivot tables to analyze:

  * TechSupport
    
  * OnlineSecurity
    
  * Streaming Services
    
* Compared churn rates for Yes vs No

---


### 5️⃣ Scatter Plot Analysis

* Compared Monthly Charges vs Total Charges
  
* Identified customer spending patterns

---

## 📈 Key Insights

* Customers with higher monthly charges tend to churn more
  
* Lack of Tech Support and Online Security increases churn
  
* Most customers fall in mid-range pricing plans
  
* Long-term customers are less likely to churn

## 📌 Conclusion

This project helps understand customer behavior and provides valuable insights to reduce churn. It demonstrates how Excel can be used effectively for data analysis and visualization.
